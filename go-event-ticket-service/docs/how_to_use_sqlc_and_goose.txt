Setting up SQLC:

1. Create a file called "sqlc.yaml" in the root of the project directory.

version: "2"
sql:
  - engine: "mysql"
    queries: "query.sql" // This is the file that contains the SQL queries with sql statements.
    schema: "schema.sql" // This is the file that contains the schema with sql statements.
    gen:
      go:
        package: "database"
        out: "internal/database"

2. Run the following command in the terminal:

sqlc generate

3. The generated files will be in the "internal/database" directory.

Setting up Goose Golang:

1. Install goose

go install github.com/pressly/goose/v3/cmd/goose@latest

2. Statements to run in the terminal:

create:
goose -dir sql/schema create <name_of_file> sql

up:
goose -dir sql/schema mysql "root:shopdev@tcp(127.0.0.1:3302)/ticket4u_test?parseTime=true" up // or up-by-one


down:   
goose -dir sql/schema mysql "root:shopdev@tcp(127.0.0.1:3302)/ticket4u_test?parseTime=true" down

status: 
goose -dir sql/schema status
